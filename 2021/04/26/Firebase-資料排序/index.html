.orderByChild() 排序規則

.orderByChild() 有一套排序規則，當今天每一筆資料的屬性內的值，若型別都不一樣時，排序規則會如下:

  1. null
  2. false
  3. true
  4. number 從小到大
  5. 字串
  6. 其他: 例如物件
