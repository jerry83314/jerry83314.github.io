express 為一款 node.js 輕量型 Web 應用式框架，可以結合資料庫做開發，包含前面的 Firebase

環境準備

當新增好資料夾之後，用終端機移動到那個資料夾底下，輸入 npm init
接著會出現 package.json 檔案，就代表成功，之後輸入 npm install express --save
就可以發現 package.json 內的 dependencies 多了 express 就成功了

開始使用 express

新增一支 JS 檔，開始引用 express

這時候可以用 console 看一下 app，如果有各種屬性跟 function 就代表是引用成功的

開啟 web 伺服器

接著就來開啟 web 伺服器

這個時候打開 localhost:5000 就可以看到 Hello World 了

使用 Express-generator 快速建立環境(2021/04/30 更新)

整合了前面幾篇講解到的東西，包含路由，EJS … 等，其實這些可以不用自己一一新增安裝
我們可以藉由 express-generator 快速建立環境
打開終端機輸入 npm install express-generator -g 安裝，且在全域
接著 cd 到 desktop，快速建立專案，輸入 express -e 專案名稱 這時候桌面上就會產生我們要的專案了
